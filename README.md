# jobs-API
## Description
```
This Jobs API application provides a RESTful API for managing jobs (jobs) and users. 
The application is implemented using Node.js for the backend and a MySQL database for data storage.
```
## Functionalities
```
- User registration with data validation.
- User login with JWT token generation.
- Adding new jobs with name, description and other relevant information.
- Deleting jobs from the database.
- Updating job information.
- Overview of all jobs and details for each job.
```
## Installation
```
npm install -> to install dependencies
npm start -> for run app
```
